# PokéOpia Favorites Tool

A lightweight HTML tool for exploring Pokémon preferences and grouped favorites.

## Features

* Search Pokémon by name
* View favorite categories and flavor preferences
* Explore grouped item relationships
* Custom group comparison tool

## How to Use

1. Download pokemon_pokopia_grouped_v100.html
2. Open `pokemon_pokopia_grouped_v100.html` in your browser
3. No installation required
